COMPUTER HARDWARE

Defination: computer hardware refers to the physical parts of a computer

categories of computer hardware include the following ; input hardware ,output hardware ,storage hardware, processing hardware, communication hardware

INPUT HARDWARE: this refers to physical componets of a computer used to enter data

examples of input hardwares include the following ; keyboard, mouse , scanner , microphones, joystick light pen touch pad etc

storage hardware : this refers to physical componets of a computer used to store data either permantely or temperarly.

examples of storage hardware include: RAM , ROM, hard disk ,flash disk ,memory cards ,CDs ,DVDs ,flopper disk

out put hardware : this refers to physical componets of a computer used to display information on a screen.

Examples of out put hardware include the following . printer , speakers , projector, monitors, etc.

processing hardware: this refers to physical componets of a computer used to manuplate data into meaningful information

Examples of processing hardware include the following . mother board, central processing unit etc.

communication HARDWARE: this refers to physical componets of a computer used in sending and recieving of data .

Examples of Communication hardware include the following . router, switch, bridge, USB, adpters etc.
